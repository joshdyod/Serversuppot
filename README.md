## What It Captures

- Hostname  
- Operating System details  
- Current Date & Time  
- Disk usage  
- Memory usage  
- CPU info  
- Kernel message logs (errors/failures)  
- Auth log failures (e.g., SSH failed logins)  
- Open TCP/UDP ports  
- Active `iptables` firewall rules
